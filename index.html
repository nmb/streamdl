Multiple streaming CORS downloads

In this example, all URL:s pasted in the box below are downloaded as a zip file output.zip by client-side javascript, using fetch calls.

Note that:

  • all URL:s must be available via CORS (instructions).
  • downloads are streamed to disk without being stored in RAM.
  • only compatible with Chrome
    • enable chrome://flags/#enable-experimental-web-platform-features
  • if a file in the list cannot be downloaded, it is added as en empty file in the zip archive
